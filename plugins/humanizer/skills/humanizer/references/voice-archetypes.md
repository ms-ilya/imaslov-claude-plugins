# Voice Archetypes

Reference points for calibration - not rigid boxes. Use the detected voice parameters from the Voice Framework to find the closest archetype, then fine-tune.

| Text type | Voice parameters | Rhythm | Fragment tolerance |
|-----------|-----------------|--------|--------------------|
| Blog / essay / opinion | Opinionated, first-person OK, hedging rare, contractions frequent | Jagged - mix 3-word and 30-word sentences freely | High - fragments, asides, tangents |
| Technical documentation | Precise, third-person, hedging minimal, contractions rare | Moderate - vary but keep readable | Low - complete sentences, but vary length |
| Email / message | Warm, direct, contractions always, hedging occasional | Short overall - few long sentences | High - fragments natural |
| Marketing / landing page | Confident, second-person, contractions frequent | Punchy - short sentences dominate | Medium |
| Academic / formal | Measured, third-person, hedging calibrated, contractions never | Moderate - longer baseline but vary | Low - but vary clause complexity |
| Social media / threads | Conversational, first-person, contractions always, hedging rare | Short and punchy - very few long sentences | Very high - fragments, abbreviations, incomplete thoughts |
| Press release / news | Formal, third-person, hedging minimal, contractions never | Moderate - longer baseline, structured | Low - complete sentences, journalistic style |
| Changelog / release notes | Direct, second-person OK, hedging never, contractions occasional | Short - one-liners dominate | Medium - fragments for minor items |
| Creative / narrative | Match the author's established voice - rhythm IS the craft | Highly variable - genre-dependent | High - fragments are a literary tool |
| Legal / compliance | Precise, third-person, hedging calibrated to liability, contractions never | Long baseline - complex clause structures expected | None - every sentence must be complete |
| Conversational / chat | Casual, first-person, contractions always, slang OK | Very short - rapid back-and-forth | Very high - anything goes |
| Mixed / unknown | Detect dominant signals, lean toward neutral | Moderate | Medium |
