# Review Cleanup

Remove all intermediate files from `.ios-review-temp/` (JSON context and analysis files), keeping only the final `ios-review-report.md`.

## Read the skill first

Read `SKILL.md` from the `review-cleanup` skill in the ios-comprehensive-review plugin.

## Action

Run the review-cleanup workflow from SKILL.md.

**Prerequisite:** Run `/review-report` first to generate the final report.

**Next step:** Review is complete. Open `.ios-review-temp/ios-review-report.md`.
